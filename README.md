Description: A machine learning-based web application that suggests movies similar to a user's choice. This project uses a content-based filtering approach, analyzing movie metadata (genres, keywords, cast, and crew) to calculate similarity scores.

Key Features:

Vectorization: Implemented Text Vectorization using Bag of Words (CountVectorizer).

Similarity Engine: Utilizes Cosine Similarity to find the closest matches between movies.

Web UI: Built an interactive dashboard using Streamlit for real-time recommendations.

Data Pipeline: Processes raw TMDB data into a compressed Similarity Matrix for fast lookup.
